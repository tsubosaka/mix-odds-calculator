# mix-odds-calculator

ポーカーのマイナーなmix gameの勝率計算アプリ

## 現在対応しているゲーム

* Razz
* Stud Hi
* Stud Hi/Lo
* 2-7 Razz
* Razzdugi
* Stud 30
* Double board Omaha Hi
* Double board Omaha Hi/Lo Best/Best

## 今後対応予定のゲーム

* Razzdeucy (2-5/2-7)

## ひょっとしたら対応

* Omaha
* Omaha Hi/Lo

## 今後追加予定機能

* テキストではなく、カード選択による入力機能
